Now that you have your NgRx in full bloom, let us take a look at what Redux in Action with Redux DevTools

Let's install Redux DevTools in three easy steps so we can take a look under the hood

  • Install browser extension
    • Chrome extension: 'redux devtools
    • Firefox add-ons: 'redux devtools'
  • Install @ngrx/store-devtools
    • 

                       > npm install @ngrx/effects --save
