# NGRX Basics

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.2.7.
This is an introduction to NGRX and state management.  This project can be viewed at <a href="https://techdoc2-ngrx.cyou" target="_blank">NGRX Basics</a>
This project also includes a basic implementation of NGRX.

## Development server
1. Download project. 
2. npm install in project directory
3. Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. 

The app will automatically reload if you change any of the source files.
